## Description du projet

L’objectif du programme est de gérer les notes des étudiants à partir d’un fichier CSV.

Le programme permet de :

- Lire les données des étudiants depuis un fichier CSV
- Calculer la moyenne de chaque étudiant
- Classer les étudiants par moyenne
- Sauvegarder les résultats dans un autre fichier CSV
